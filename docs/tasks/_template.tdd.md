## 验证命令

```bash
pnpm typecheck
# 后端任务
pnpm --filter @agent/api lint
# 前端任务
pnpm --filter @agent/web lint
```

根据改动范围选择后端或前端 lint；只有全仓 lint 基线已清理时，才把 `pnpm lint` 作为强制验收。

如果任务涉及数据库，补充：

```bash
pnpm prisma:generate
pnpm exec prisma validate
```

## 任务状态

- 实施状态：未开始 / 进行中 / 已实现

只有“实施状态：已实现”且“验收状态：已通过”时，任务才可以进入 Completed。
